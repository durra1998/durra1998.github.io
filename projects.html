Quantum-Classical Hybrid Protocol for Secure Image Encryption

This encryption technique is implemented in the context of a hybrid quantum-classical cryptosystem in which a Quantum Key Distribution protocol (D Pastorello, 2017) is combined with the Logistic Sine map to establish a secure image transmission scheme. Both communication parties utilize same initial values and control parameter for successful encryption and decryption. This encryption method exhibits high levels of reliability and resilience. The project is available here.

Key Features:

  • Provides provable security against quantum computing attacks.
  • Uniformly distributes the pixel intensities across the image, effectively masking the statistical characteristics of the original image.
  • Maintains computational efficiency for practical deployment.
  • Demonstrates resistance to common cryptographic attacks.

Simulation of a Quantum Private Query Protocol Using W-States in Qiskit with Noise Analysis

This project implements and simulates a Quantum Private Query (QPQ) protocol utilizing entangled W-states (RG Zhou, 2021) in the context of the Noisy Intermediate-Scale Quantum (NISQ) era, where quantum systems exhibit non-negligible noise levels. Realistic noise models, including gate and measurement errors, are integrated into the simulation to evaluate how well the protocol performs under practical conditions. The project is available on my GitHub.

Heatmap showing how protocol success varies with noise levels.

Key Features:

  • Quantum Private Query Protocol enables secure database queries without revealing user intent.
  • NISQ-Era Feasibility Analysis explores the robustness and practicality of QPQ on current quantum devices.
  • Measurement noise poses a greater challenge to protocol reliability than gate noise.
  • The protocol demonstrates robustness under low to moderate noise, highlighting its promise for NISQ-era devices.

Design an Inset-fed Microstrip Antenna at 2.4 GHz using ADS

This project involved the design and simulation of an inset-fed rectangular microstrip patch antenna tuned to operate at 2.4 GHz using Advanced Design System (ADS), a powerful RF/microwave simulation tool. The antenna structure was optimized to achieve impedance matching and minimal signal reflection by adjusting the inset feed position and patch dimensions. Key performance metrics such as the reflection coefficient (S11), Smith chart, surface current distribution, and 2D radiation pattern were analyzed. This was completed as part of an undergraduate project for my L-3, T-1 Microwave and Antenna Engineering sessional.
